Xero-connected reports · optional

Populate the workbook with your Xero data.

Invite Lyros as an advisor. We pull your data through the Xero and Microsoft APIs, write it straight into the workbook, and send the XLSX back.

No LLM touches your finance data.

This is a direct API pipeline. The Xero API for the data, the Microsoft Graph API to write it into Excel. Both are stable, audited, decade-old APIs. Nothing is sent to a model. Nothing is stored beyond the run.
